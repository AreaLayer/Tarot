Taro web wallet

Cryptocurrency wallet for the taro protocol. Trade and mint altcoins and NFTs directly on Bitcoin blockchain.

Our Mission

User friendly Taro wallet on testnet

Taro protocol is an on-chain protocol that allows for the minting of alternative currencies, stablecoins and collectibles to be recorded on the Bitcoin blockchain.

Taro wallet gives users a web interface to interract with taro deamon running on Bitcoin testnet without the need to operate a Lightning node.

Status of Taro

Taro wallet is still in Alpha

Taro wallet is based on tarod deamon. Tarod is still an experimental software with bugs often causing loss of all funds. The wallet software itself is in Alpha.
